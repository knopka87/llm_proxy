Извлеки печатный текст условия с фото и минимальную структуру задачи.

Верни ТОЛЬКО JSON по PARSE.response.v1:
{
  "raw_task_text": "<дословно печатный текст; рукопись игнорируется; закрытые рукописью буквы заменены на '_'; варианты в скобках сохраняются дословно>",
  "task_struct": {
    "subject": "<math|russian|generic>",
    "type": "<краткая метка, если понимаешь (например: grammar|arithmetic|roman_numerals|fill_blanks|table|list_by_lines|visual_compare)>",
    "combined_subpoints": true
  },
  "needs_user_confirmation": <true|false>
}

Правила:
- Сохраняй переносы строк/пробелы и визуальную структуру (строки, таблицы, колонки).
- Рукописные дополнения не включай. Если буква печати закрыта рукописью — ставь "_".
- Курсив и наклонный шрифт тоже считаются печатным текстом и должны попадать в raw_task_text.
- В заданиях с вариантами букв/слов в скобках сохраняй скобки и все варианты дословно.
- Если смысл задания в основном в рисунке/схеме (сравнение длины, количества и т.п.), используй type:"visual_compare" и ставь needs_user_confirmation:true.
- Не реконструируй числовые ряды/списки — передавай только то, что напечатано.
- Не добавляй подсказок/комментариев, не выводи ничего кроме JSON.
