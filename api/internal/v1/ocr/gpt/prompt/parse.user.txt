Извлеки печатный текст задания с изображения и минимальную структуру задачи.

Верни ТОЛЬКО JSON по PARSE.response.v1:
{
  "raw_task_text": "<дословно печатный текст; вся рукопись ИГНОРИРУЕТСЯ; если рукопись закрыла печатную букву/цифру — поставь '_' на её месте; сохрани переносы строк/пробелы (каждый пример с новой строки)>",
  "task_struct": {
    "subject": "<math|russian|generic>",
    "type": "<arithmetic|grammar|roman_numerals|fill_blanks|list_by_lines|table|... (если понимаешь)>",
    "combined_subpoints": true
  },
  "needs_user_confirmation": <true|false>
}

Правила:
- Сохраняй переносы строк/пробелы и визуальную структуру (строки, таблицы). Разные примеры — с новой строки.
- Рукописные дополнения НЕ включать. Если буква печати закрыта рукописью — ставь "_".
- Не «угадывай» пропущенные символы: только печать, рукопись игнорируется.
- Не переопределяй предмет; если неочевидно — укажи subject: "generic".
- Никакого текста до/после JSON.

ПРИМЕРЫ, ЧТО НЕЛЬЗЯ ПЕРЕНОСИТЬ (негативные):
– Математика: вписанные от руки цифры в квадратиках/клетках/под чертой — НЕ переносить (оставь "__" или "_").
– Таблица римские↔арабские: ручные "50, 500, 201, 72" и ручные X/C/D/M/LC — НЕ переносить. Печатные V, L, C, D, M, CCIX, MDLX и числа 5,10,100,1000 — оставить.
– Русский с пропусками: если рукопись закрыла печатную букву — ставь "_": "Поз_ний зв_нок".
– Ребусы/таблицы-сортировки: подписи ребёнка под картинками, слова в полях — НЕ переносить.
