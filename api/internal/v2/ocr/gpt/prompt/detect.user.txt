Определи предмет задания на фото.

Верни ТОЛЬКО JSON по DETECT.response.v1:
{
  "subject_hint": "<math|russian|generic>",
  "grade_hint": <1..4, опционально>,
  "confidence": <0..1>,
  "debug_reason": "<краткая причина, опционально>"
}

Правила:
- Никогда не оставляй subject_hint пустым: выбери лучший из {math,russian,generic}.
- Если картинка нечитабельна, возвращай "generic" с низкой confidence.
- Не добавляй никакого текста до/после JSON.
