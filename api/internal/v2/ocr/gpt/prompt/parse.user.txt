Извлеки печатный текст условия с фото и минимальную структуру задачи.

Верни ТОЛЬКО JSON по PARSE.response.v1:
{
  "raw_task_text": "<дословно печатный текст; рукопись игнорируется; закрытые рукописью буквы заменены на '_' >",
  "task_struct": {
    "subject": "<math|russian|generic>",
    "type": "<краткая метка, если понимаешь (например: grammar|arithmetic|roman_numerals|fill_blanks|table|list_by_lines)>",
    "combined_subpoints": true
  },
  "needs_user_confirmation": <true|false>
}

Правила:
- Сохраняй переносы строк/пробелы и визуальную структуру (строки, таблицы).
- Рукописные дополнения не включать. Если буква печати закрыта рукописью — ставь "_".
- Не добавляй подсказок/комментариев, не выводи ничего кроме JSON.
